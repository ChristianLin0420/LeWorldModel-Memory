## Decisions for `paper_d`

1. Each main figure answers one question: discovery, causal necessity, official-host use, or host readiness.
2. The architecture uses four distinct raw observations and four verbs: **DISCOVER → STORE → VERIFY → RECALL**.
3. All figure and axes backgrounds are pure white. Yellow means proposed/surprising, green means verified, red means rejected, and grey means frozen/control.
4. Main results use balanced accuracy, overwrite/retrieval rates, normalized prediction improvement, and normalized deletion effects. Raw MSE and implementation diagnostics remain in the appendix.
5. Causal evidence is always a matched deletion contrast through the frozen host. Attention, surprise, or probe readability alone is not called causal.
6. The raw-frame selector is algorithmic and auditable. Frames maximize pixel change after a minimum temporal separation; object crops and difference scores make small 64×64 changes visible.
7. LeWM host readiness stays outside the main CEM pipeline because storage and host injection are distinct questions.
