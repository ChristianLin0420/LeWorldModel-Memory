# ICLR LaTeX build

ICLR-format LaTeX source for the paper, generated from [`../docs/ICLR.md`](../docs/ICLR.md).

## Files
- `main.tex` ‚Äî document shell (title, abstract, `\input`s, bibliography).
- `iclr2027_conference.sty` ‚Äî ICLR-style format (Times, running header, ruled abstract, numbered sections).
- `build_paper.py` ‚Äî converts `../docs/ICLR.md` ‚Üí `abstract.tex`, `body.tex`, `refs.tex`.
- `abstract.tex` / `body.tex` / `refs.tex` ‚Äî generated fragments (regenerated by `build_paper.py`).
- `figures/` ‚Äî figures (copied from `../docs/figures/`).
- `main.pdf` ‚Äî the compiled paper (17 pages).

## Rebuild
The Markdown‚ÜíLaTeX step tokenizes all math/unicode to inert placeholders, runs `pandoc`, then
restores LaTeX ‚Äî this sidesteps pandoc mangling `$...$`/`\(...\)`. Figures/equations become raw
LaTeX blocks; figure captions are unnumbered (`\caption*`) so their literal "Figure N" labels
stay in sync with the in-text references.

```bash
# 1. regenerate LaTeX fragments from the Markdown (needs pandoc on PATH)
python build_paper.py
# 2. refresh figures if docs/figures changed
cp ../docs/figures/*.png figures/
# 3. compile (needs a TeX distribution with latexmk, e.g. TinyTeX)
latexmk -pdf -interaction=nonstopmode main.tex
```

Toolchain used: TinyTeX (TeX Live 2026) + pandoc 3.10. Required TeX packages: `titlesec`,
`booktabs`, `longtable`, `caption`, `times`/`mathptmx`, `hyperref`, `microtype`, `calc`, `rsfs`,
`courier` (plus the usual `amsmath`/`graphicx`).
